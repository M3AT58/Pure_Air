# بيور إير - موقع خدمات شاملة للمنازل والمؤسسات (Pure Air Website)

موقع ويب شامل لشركة "بيور إير" المتخصصة في مكافحة الحشرات وعزل وتعقيم الخزانات ونظافة وصيانة المكيفات، مطور باستخدام Laravel 12 مع دعم كامل للغة العربية.

## المميزات

- **واجهة باللغة العربية**: الموقع مطور بالكامل باللغة العربية مع دعم RTL
- **خدمات متنوعة**: مكافحة الحشرات، عزل وتعقيم الخزانات، نظافة وصيانة المكيفات
- **نظام إدارة الصور المتقدم**: صور متعددة للخدمات وصور بارزة للمقالات من Unsplash
- **مدونة متكاملة**: نظام إدارة المحتوى مع مقالات متخصصة مع صور جذابة
- **تصميم متجاوب**: يعمل بشكل مثالي على جميع الأجهزة مع carousel تفاعلي
- **لوحة تحكم إدارية**: إدارة شاملة للمحتوى والخدمات والاستفسارات مع رفع الصور
- **تحسين محركات البحث**: SEO محسّن للمحتوى العربي مع الصور المحسنة

## التقنيات المستخدمة

- **Backend**: Laravel 12
- **Frontend**: Bootstrap 5 RTL + Font Awesome
- **Database**: SQLite (التطوير) / MySQL (الإنتاج)
- **Fonts**: Cairo و Tajawal للنصوص العربية
- **Styling**: CSS مخصص مع دعم RTL

### فئات الإعدادات

#### 1. إعدادات الموقع الأساسية
- `site_title` - عنوان الموقع
- `site_description` - وصف الموقع
- `site_keywords` - كلمات مفتاحية للSEO
- `hero_title` و `hero_subtitle` - محتوى القسم الرئيسي

#### 2. معلومات الشركة والاتصال
- `contact_phone`, `contact_email`, `contact_address`
- `whatsapp_number`, `instagram_handle`
- `emergency_phone` - هاتف الطوارئ
- `service_areas` - مناطق الخدمة
- `company_hours` - ساعات العمل

#### 3. إعدادات الصور
- `default_service_image` - صورة افتراضية للخدمات
- `default_blog_image` - صورة افتراضية للمدونة
- `image_quality` - جودة الصور (80%)
- `max_images_per_service` - عدد أقصى للصور لكل خدمة (5)
- `enable_image_lazy_loading` - تفعيل التحميل التدريجي
- `image_alt_prefix` - بادئة النص البديل للصور

#### 4. إعدادات الخدمات والمدونة
- `services_per_page` - عدد الخدمات لكل صفحة (9)
- `featured_services_count` - عدد الخدمات المميزة (3)
- `enable_service_carousel` - تفعيل عرض الصور المتدرج
- `posts_per_page` - عدد المقالات لكل صفحة (6)
- `enable_featured_images` - تفعيل الصور المميزة

#### 5. إحصائيات الشركة
- `years_experience` - سنوات الخبرة (10)
- `satisfied_customers` - العملاء الراضون (1000)
- `completed_projects` - المشاريع المنجزة (2500)
- `certified_technicians` - الفنيون المعتمدون (15)

#### 6. إعدادات العلامة التجارية
- `primary_color` - اللون الأساسي (#28a745)
- `secondary_color` - اللون الثانوي (#1e7e34)
- `accent_color` - لون التمييز (#c3e6cb)
- `brand_logo_url` - رابط الشعار
- `brand_favicon` - أيقونة الموقع

### نظام Helper للإعدادات

#### استخدام Helper Class
```php
use App\Helpers\SettingsHelper;

// الحصول على إعداد واحد
$title = SettingsHelper::get('site_title', 'افتراضي');

// الحصول على عدة إعدادات
$settings = SettingsHelper::getMultiple(['site_title', 'contact_phone']);

// الحصول على إعدادات الشركة
$companyInfo = SettingsHelper::getCompanyInfo();

// الحصول على إعدادات الصور
$imageSettings = SettingsHelper::getImageSettings();

// التحقق من تفعيل ميزة
$isEnabled = SettingsHelper::isFeatureEnabled('enable_service_carousel');

// الحصول على إعداد رقمي
$postsPerPage = SettingsHelper::getNumeric('posts_per_page', 6);
```

#### استخدام Helper Functions (في المعالجات والViews)
```php
// الحصول على إعداد
$title = setting('site_title');

// معلومات الشركة
$company = company_info();

// إعدادات الصورة الافتراضية
$defaultImage = default_service_image();

// التحقق من تفعيل ميزة
$carouselEnabled = is_feature_enabled('enable_service_carousel');

// إعداد رقمي
$experience = numeric_setting('years_experience', 0);
```

### نظام إدارة الصور المحسن

#### الصور للخدمات
- كل خدمة تدعم حتى 5 صور متعددة
- عرض الصور في carousel تفاعلي
- صور عالية الجودة من Unsplash (800x600)
- تحميل تدريجي لتحسين الأداء

#### الصور للمدونة
- صورة مميزة لكل مقال
- عرض تلقائي في قوائم المقالات
- صور احترافية مناسبة للمحتوى

#### إعدادات الصور المتقدمة
```php
// جودة الصور
$quality = numeric_setting('image_quality', 80);

// نسبة العرض للصور
$aspectRatio = setting('service_image_aspect_ratio', '4:3');

// بادئة النص البديل
$altPrefix = setting('image_alt_prefix', 'بيور إير -');
```

### نظام التخزين المؤقت (Caching)
- تخزين مؤقت ذكي للإعدادات (3600 ثانية)
- تحديث تلقائي للذاكرة المؤقتة عند تغيير الإعدادات
- تحسين الأداء للوصول المتكرر للإعدادات

```php
// مسح ذاكرة الإعدادات المؤقتة
SettingsHelper::clearCache();
```

### إدارة الإعدادات

#### إضافة إعداد جديد
```php
// في SettingSeeder.php
['key' => 'new_setting', 'value' => 'default_value', 'type' => 'text']

// أو برمجياً
SettingsHelper::set('new_setting', 'value', 'text');
```

- `email` - بريد إلكتروني
- `url` - رابط
- `number` - رقم
- `boolean` - صحيح/خطأ
- `color` - لون
